清末，英、法、俄等八国强硬侵压中国，日本亦对中国侵略不断。天津霍元甲志存高远的他，练得一身好功夫。霍元甲有两挚友，农劲荪和王五。三人各有抱负，霍元甲认为“欲使国强，非得人人习武”。

霍元甲父霍恩第与师弟赵声显却因“秘宗拳”正宗，三十年走过来，两家械斗不断，霍元甲与赵家女儿赵倩男，迫得在感情事上不得不暂且划上句号。霍、赵两家一年一度比赛争取“正宗”头衔的日子到了，中途横出一独臂老人程天笑，此次偕徒弟陈真回来搅局，霍元甲把程天笑打走。程天笑性格偏激，受不了输在一个后辈小子手上，跟霍元甲再次比武3次，最后还是用了秘宗拳的最后3式把程天笑打死。陈真誓杀霍元甲为师报仇。
